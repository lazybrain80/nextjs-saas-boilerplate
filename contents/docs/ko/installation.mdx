1. 템플릿을 다운로드하고 압축을 풉니다. 그런 다음 해당 디렉터리로 이동하여 다음 명령을 실행하여 종속 요소를 설치합니다:
```
npm install
```
2. 설치가 완료되면 다음 명령을 실행하여 개발 서버를 시작합니다:
```
npm run dev
```

이 명령이 성공적으로 실행되면 템플릿이 localhost:3000에서 열립니다. 이제 브라우저에서 모든 변경 사항을 확인할 수 있습니다.

## 서버에 배포하기

프로젝트를 서버에 배포하려면 먼저 프로젝트를 빌드해야 합니다. 다음 명령을 실행하여 프로젝트를 빌드합니다:
```
npm run build
```
빌드가 완료되면 다음 명령을 실행하여 서버를 시작합니다:
```
npm run start
```
이제 프로젝트가 서버에서 실행됩니다. 브라우저에서 서버의 IP 주소를 입력하면 서버에 액세스할 수 있습니다.

## Vercel에 배포하기

Vercel에 프로젝트를 배포하려면 GitHub 계정을 Vercel에 연결해야 합니다. 계정을 연결한 후 배포 버튼을 클릭하여 프로젝트를 배포할 수 있습니다.
