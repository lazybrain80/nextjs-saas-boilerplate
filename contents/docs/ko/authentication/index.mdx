이 문서에서는 NextAuth를 구성하고 인증을 활성화하는 방법을 보여드리겠습니다.

### Supabase 인증

Supabase는 개발자가 사용자 지정 인증 시스템을 처음부터 구축하는 번거로움을 덜어주는 다양한 인증 서비스를 제공합니다.
Supabase 인증은 소셜 인증 및 타사 제공업체를 사용한 전화 인증 등 널리 사용되는 여러 인증 방법과 함께 작동합니다.
지원되는 타사 제공업체 목록은 다음 [Supabase 문서](https://supabase.com/docs/guides/auth)를 참조하세요.

  - [Supabase 웹사이트](https://supabase.io/)로 이동하여 계정을 등록합니다.
  - 새 프로젝트를 생성하고 'API URL'과 '아논 키'를 메모합니다.
