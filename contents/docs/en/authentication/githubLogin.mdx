## 1. How to create an OAuth application in GitHub Developer Settings

1. go to [GitHub Developer Settings](https://github.com/settings/developers).
2. click `New OAuth App` button.

<Image
  src='/images/mdx/github_user_auth_0.png'
  alt='github_user_auth_0'
  width={1024}
  height={768}
  caption='github - user auth'
/>

3. Enter the application name, homepage URL, and authorised redirect URI.
  - Homepage URL: `http://localhost:3000`
  - Authorised redirect URI: `https://your-supabase-url/auth/v1/callback`
  - Leave ‘Device Flow’ unchecked.
4. Copy and enter the new OAuth credentials into the Supabase dashboard.
  - Copy and save the Client ID.
  - Click Generate new client password.
  - Copy and save the client password.

## 2. Set up Google OAuth
1. go to `Authentication`- `Providers` in the [Supabase](https://supabase.io) project dashboard.
2. select `GitHub` in the `Providers` section.

<Image
  src='/images/mdx/github_user_auth_1.png'
  alt='github_user_auth_1'
  width={1024}
  height={768}
  caption='github - user auth'
/>

3. Enter the client ID and secret key created in the OAuth application in GitHub Developer Settings into Supabase.
