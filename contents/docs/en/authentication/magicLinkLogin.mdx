## Setting up Email login (MagicLink) in Supabase

1. go to your Supabase project
    - Log in to the [Supabase](https://supabase.io) website.
    - Select a project and navigate to the `Authentication` tab in the dashboard.

2. Set up authentication
    - In the `Settings` tab, set the `Site URL`. This is the URL that users will be redirected to when they log in via an email link.
    - In the `Provider - Email` section, enable the `Enable email provider` option.
